# Temple Background Image

## Current Setup

- The app is currently using a beautiful temple image from Unsplash
- This provides a stunning background for the authentication screens

## To Use Your Own Image:

1. Replace the NetworkImage URL in `/lib/Screens/auth_screen.dart`
2. OR add your own image file here as `temple_bg.jpg` or `temple_bg.png`
3. Then update the code to use: `AssetImage('assets/images/your_image.jpg')`

## Recommended Image Specifications:

- **Format**: JPG or PNG
- **Resolution**: 1920x1080 or higher
- **Aspect Ratio**: 16:9 or 4:3
- **File Size**: Under 2MB for optimal performance

## Example Images That Work Well:

- Temple or church silhouettes
- Sunrise/sunset temple views
- Architectural church details
- Spiritual or serene landscapes
